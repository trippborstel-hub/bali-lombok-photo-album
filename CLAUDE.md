# Bali & Lombok Photo Album

A simple, static photo gallery site for sharing travel photos with family.

## Stack

- Plain HTML/CSS/JS — no build tools or frameworks
- Single `index.html` with inline styles and script
- Masonry-style responsive gallery with lightbox

## Structure

- `index.html` — main page
- `photos/` — optimized images for the gallery (to be created)
- `_reference/` — original/source photos (not served directly)

## Guidelines

- Keep it simple — no dependencies, no build step
- Prioritize fast load times (lazy loading, optimized images)
- Mobile-friendly responsive layout
